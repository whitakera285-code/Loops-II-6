Write a function called `printCoordinates` that takes in one parameter, the `size` of the grid, and prints to the console all the x, y coordinates for a square grid of that size. Note: both your x and y should start at 0!

For example:

```
> printCoordinates(3)
```

```
(0, 0)
```

```
(0, 1)
```

```
(0, 2)
```

```
(1, 0)
```

```
(1, 1)
```

```
(1, 2)
```

```
(2, 0)
```

```
(2, 1)
```

```
(2, 2)
```
